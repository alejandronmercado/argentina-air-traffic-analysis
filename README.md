📊 Argentina Air Traffic Analysis (2020–2025)
📌 Overview

This project analyzes commercial aviation data in Argentina between 2020 and 2025, focusing on passenger demand, operational efficiency, and market distribution.

🧰 Tools
- **Power BI** - Dashboard visualization
- **Python / Pandas** - Data cleaning and EDA
- **Git & GitHub** - Version control

Data obtained from datos.gob.ar, including over 500,000 records with:

Airlines
Origin and destination (provinces)
Passengers
Available seats
Flights

🔧 Data Processing
Data cleaning and preprocessing
Date transformation for time analysis
Data modeling for dashboard visualization

📈 Key Analysis
Passenger evolution over time
Market share by airline
Geographic distribution of passengers
Domestic vs international flights
Monthly seasonality

🔍 Key Insights
Strong post-pandemic recovery (7M → 29M passengers)
High occupancy rate (84%) indicating efficient capacity usage
Market dominance by Aerolíneas Argentinas
Strong concentration in Buenos Aires region
Domestic flights represent ~65% of total traffic
Clear seasonal patterns driven by tourism
